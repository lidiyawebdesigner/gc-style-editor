# GetCourse Style Editor

Браузерное расширение для Chrome, которое позволяет красиво и быстро стилизовать блоки в **GetCourse** — без знания CSS. Кликнул блок → настроил в панели → скопировал чистый CSS → вставил в блок. Работает офлайн, без AI и без API-ключей.

> Внутренний инструмент дизайн-направления Zerocoder. Финальный проект практического курса по Claude Code.

---

## Проблема

Чтобы оформить блок в GetCourse аккуратно, нужно править CSS-стили прямо внутри платформы. Встроенный редактор для этого неудобен, а **без знания CSS сделать красиво невозможно**. В итоге дизайнеры и тех-специалисты либо тратят время на ручную возню с кодом, либо оставляют шаблонный вид.

## Решение

Расширение даёт визуальную панель прямо на странице GetCourse:

1. **Кликаешь блок** — он подсвечивается, в боковой панели появляются его параметры.
2. **Настраиваешь** — цвета, отступы, тени, радиусы, типографика, градиент, прозрачность; готовые пресеты; детальная стилизация форм.
3. **Копируешь чистый CSS** — расширение собирает семантический экспорт по эталонным селекторам GetCourse.
4. **Вставляешь** в CSS-поле блока GC — стиль применяется на проде, остальная страница не ломается.

## Для кого

Дизайн-отдел и тех-специалисты **Zerocoder**, которые верстают и оформляют блоки в GetCourse.

## Что умеет

- **Click-to-select** — выбор любого блока кликом, подсветка элемента.
- **Контролы стилей** — фон/текст (color picker), отступы, скругления, тени (включая лаймовое свечение), градиент, прозрачность.
- **Типографика** — размер, межстрочный, жирность, выравнивание, регистр (Aa / КАПС / aa).
- **Детальная стилизация форм GC** — поля ввода, плейсхолдер, фокус, автозаполнение, согласие, кнопка — по эталонным селекторам GetCourse (`.lt-form .f-input`, `.btn.f-btn`, `.global-confirm-checkbox-block` и др.).
- **Пресеты** — готовые стили в один клик.
- **Библиотека готовых адаптивных блоков** — self-contained HTML-блоки со scoped-CSS для вставки в GetCourse.
- **Чистый семантический экспорт** — 5–6 правил вместо 95, без ID-каши и мусора; рукой можно поправить.
- **Mirror-to-siblings** — применить стиль сразу ко всем колонкам/карточкам блока.
- **Встроенный канал обратной связи** — заметки прямо из панели уходят в рабочий процесс.

## Установка (режим разработчика)

1. Открой `chrome://extensions/`.
2. Включи **Режим разработчика** (тумблер справа сверху).
3. Нажми **Загрузить распакованное расширение** (Load unpacked).
4. Выбери папку **`extension/`** (именно вложенную, не корень проекта).
5. Открой любую страницу `*.getcourse.ru` — справа появится панель.

После изменений кода: `chrome://extensions/` → **↻** на расширении → перезагрузи страницу GetCourse.

## Технологии

- Vanilla JavaScript, без фреймворков и зависимостей
- Chrome **Manifest V3**
- Content script + инжектируемый сайдбар (без отдельного popup)
- Без бэкенда, без API-ключей, без AI в рантайме — всё работает локально

## Структура проекта

```
extension/        — код расширения (загружается в Chrome как unpacked)
  manifest.json   — конфиг
  content.js      — панель + логика стилизации и экспорта
  templates.js    — библиотека готовых блоков
  sidebar.css     — стили панели
.claude/
  skills/         — скилл-оркестратор команды разработки
  agents/         — команда субагентов (идеи, дизайн, CSS, ревью, тест)
tools/            — служебные dev-скрипты
```

## Как это построено

Расширение разрабатывается мультиагентным конвейером Claude Code: отдельные агенты на идеи фич, дизайн панели, рабочий CSS, ревью и тест — с гейтами качества между этапами. Конвейер описан в `.claude/skills/gc-pipeline/`.

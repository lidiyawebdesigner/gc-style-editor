# Скриншоты

Сюда кладутся скриншоты работающего расширения для README и сдачи проекта.

Рекомендованный шот-лист:
- `panel.png` — боковая панель расширения на странице GetCourse.
- `before-after.png` — блок до и после стилизации.
- `form-styling.png` — настройка формы (поля, кнопка).
- `export.png` — окно «Скопировать стили» с готовым CSS.
- `library.png` — библиотека готовых блоков.

После добавления — раскомментируй соответствующие строки `![...]()` в корневом `README.md`.
